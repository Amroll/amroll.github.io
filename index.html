NSIS插件开发(1)

写在前面

本列系主要说NSIS插件上开发的一些事情，主要是本人自己在开发的过程中碰到的一些问题的解决。

NSIS

NSIS(Nullsoft Scriptable Install System)是一个开源的 Windows 系统下安装程序制作程序。它提供了安装、卸载、系统设置、文件解压缩等功能。这如其名字所指出的那样，NSIS 是通过它的脚本语言来描述安装程序的行为和逻辑的。NSIS 的脚本语言和通常的编程语言有类似的结构和语法，但它是为安装程序这类应用所设计的。

开发准备

开发工具

本人用的是Visual Studio 2015 Communtiy。

头文件与库

先安装完NSIS后，在安装目录下/Examples/Plugin/nsis文件夹下api.h, nsis_tchar.h, pluginapi.h, pluginapi-x86-ansi.lib, pluginapi-x86-unicode.lib都为开发需求文件。

一步一步

  1. 新建项目
  2. 设置项目->属性->C/C++->代码生成->运行库这一项时，在DEBUG模式下为多线程调试(/MTd)，在RELEASE模式下为多线程(/MT)

IWB/CFF 学习笔记

IWB/CFF

IWB : Interatactive Whiteboard File Format
CFF : Common File Format

名词解释

File 电脑文件
Document 文件包含多个页面
Page elements放置的地方
Slide Page显示的地方
Elements Page上显示的对象
Reader 程序打开文件
Writer 保存
SVG Scalar Vetor Graphics Format
SVGT SVG Tiny specification

引用

XML 1.0 http://www.w3.org/TR/REC-xml/
SVG 1.1 http://www.w3.org/TR/SVG11/
SVG 1.2 (draft) http://www.w3.org/TR/2004/WD-SVG12-20041027/
SVG Tiny 1.2 (draft) http://www.w3.org/TR/SVGMobile12/
CSS2 http://www.w3.org/TR/REC-CSS2/cover.html

文件格式

后缀

IWB文件后缀.iwb

主体

IWB文件是一个ZIP的压缩文件。
注1： 如果IWB文件可以用XML完全的记录，那么ZIP可以只有一个XML文件，文件名为：content.xml
注2： 如果不能用XML完全的描述，如：包含图片，视频等信息。那么这些文件也要一起打包进ZIP文件里面，同时也只有一个content.xml文件来描述IWB的文档。
